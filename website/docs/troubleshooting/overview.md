# Troubleshooting

Entry point for diagnosing common Shaktiman issues. Match your symptom to the relevant
page:

| Symptom | Page |
|---|---|
| `shaktimand` won't start | Daemon & leader election |
| Search returns nothing | Empty or bad results |
| Results are stale | Indexing stuck |
| Embedding never finishes | Embedding failures |
| Queries are slow | Performance problems |
| Postgres / Qdrant connect errors | Backend errors |

:::note Placeholder

Each sub-page follows a **Symptom → Likely causes (ranked) → Diagnostic commands → Fix**
template. Authored in Step 5b.

:::
