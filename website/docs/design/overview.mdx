# Design & ADRs

This section collects the architectural design record for contributors. It is **not**
user-facing documentation — pages here describe design intent, some of which has diverged
from the shipped system. Every imported page carries a "Status (Today)" note so readers
know whether the design reflects current code.

Canonical references:

- `docs/architecture/03-architecture-v3.md` — design intent for the query engine,
  storage, and enrichment pipeline.
- `docs/architecture/03-architecture-v3-status.md` — shipped-vs-designed reconciliation.
- `docs/design/adr-00{1,2,3,4}-*.md` — accepted architectural decisions.
